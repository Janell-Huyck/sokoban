Empty Tummy!

Oh no! You made it home through the woods... and now you're starving!

Push ALL the empty honey jars onto the honey combs to get a scrumptious treat!

Use the arrow keys or W,A,S,D to push the jars onto the honeycombs.

Congratulations! You Got All the Honey!

How To Play

  • Object of the game is to cover all honeycomb with jars.
  • Move the bear by using either W,A,S,D keys or up/down/right/left arrow keys.
  • Push jars by attempting to move the bear onto them.
  • You can only push jars, not pull them.
  • If something is behind a jar, you cannot push it.
  • Use the "Start Over" button to reset the map if you get stuck.
  • Good Luck!
